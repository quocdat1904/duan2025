# Dự án Kết nối Việt Nam - Đầy đủ Frontend + Backend

## 📁 Cấu trúc
- `client/` — Giao diện React (Vite). Đã build sẵn ở `client/dist`
- `server/` — Backend Node.js + Express + MongoDB

## ▶️ Cách chạy local
### 1. Chạy backend:
```
cd server
npm install
node index.js
```

### 2. Chạy frontend:
```
cd client
npm install
npm run dev
```

Hoặc deploy thư mục `client/dist` nếu chỉ cần tĩnh.
